1	БІЗНЕС ВИМОГИ
1.1		Передумови

У сфері відпочинку та розваг, зокрема на гірськолижних курортах, безпека та підготовка персоналу стають важливими факторами для забезпечення комфорту та безпеки гостей. Керування тренуваннями та забезпеченням безпеки на курорті є ключовими елементами стратегії з управління ризиками та підвищення якості обслуговування для відвідувачів.

1.2		Бізнес можливості

Аналіз ринку:
Програмний продукт призначений для корпоративних клієнтів у гірськолижній галузі для автоматизації бізнес-процесів, а для звичайних гостей курортів – забезпечення комфортних умов та продуктивного і безпечного тренування. На ринку існують компанії, які також надаються послуги в сфері автоматизації гірнолижних курортів, такі як наприклад
Inntopia, яка пропонує інтегровані програмні рішення для гірськолижних курортів, включаючи системи бронювання, квиткові платформи, аналітику та інші інструменти для управління різними аспектами курортного бізнесу, а також MountainGuard – компанія, яка спеціалізується на розробці та впровадженні систем безпеки для гірськолижних курортів, включаючи системи відеоспостереження, системи трекінгу гірських трас, системи аварійного сповіщення та інші технологічні рішення для підвищення безпеки на курортах.

2	КОНЦЕПЦІЯ РІШЕННЯ
2.1		Окреслення концепції

SnowWarden дасть можливість в реальному часі відстежувати місця та траси, де проводяться тренування, контролювати кількість та розподіл гірськолижників, а також надавати швидку реакцію на будь-які надзвичайні ситуації або аварії. Крім того, система забезпечить зручний інтерфейс для користувачів, що дозволить їм легко відстежувати тренування, замовляти послуги та отримувати актуальну інформацію про стан безпеки на курорті. Основною метою системи є підвищення рівня безпеки для гостей курорту, оптимізація управління персоналом та забезпечення позитивного досвіду відвідувачів. Крім цього, вона передбачатиме впровадження програми лояльності, яка стимулюватиме постійних клієнтів та надаватиме їм додаткові переваги за активне використання послуг курорту. Впровадження такої системи дозволить курортам покращити свою конкурентоспроможність, привернути нових клієнтів та збільшити загальний прибуток.

2.2		Головна функціональність

MF-1: Надання можливості зручного відстеження тренувань та забезпечення безпеки на гірськолижному курорті.
MF-2: Можливість записуватись на тренування.
MF-3: Можливість залишати відгуки до трас і тренерів.
MF-4: Відображення актуальної інформації про траси, розташування тренерів та гірськолижників у реальному часі для оптимізації управління ресурсами.
MF-5: Реалізація програми лояльності для активних користувачів, що надає додаткові переваги та стимулює постійних відвідувачів курорту.
MF-6: Моніторинг умов безпеки на трасах та сповіщення в разі виникнення небезпеки або аварійних ситуацій.
MF-7: Планування та оптимізація маршрутів для гірських рятувальних служб для швидкого реагування на надзвичайні ситуації.

4	БІЗНЕС-КОНТЕКСТ
4.3		Робоче середовище

-	Мова програмування: Ваша система буде розроблена з використанням мови програмування C#. Ця мова дозволить створювати ефективний та надійний код для різних складних функцій системи.
-	База даних: Для зберігання даних вашої системи використовуватиметься PostgreSQL, що забезпечить надійність, швидкодію та гнучкість у взаємодії з даними. Адміністрування бази даних буде здійснюватися через інтерфейс PgAdmin4.
-	Серверна частина: Для розробки серверної частини вашого додатку буде використаний фреймворк ASP.NET Core. Це забезпечить високу ефективність, масштабованість та безпеку вашої системи.
-	Веб-застосунок: Для створення веб-інтерфейсу вашого додатку буде використана бібліотека React, яка забезпечить швидкість розробки та динамічність веб-сторінок. Ви також використовуватимете JavaScript, TypeScript, HTML, CSS та SCSS для створення інтерактивних інтерфейсів.
-	Мобільний застосунок: Для розробки мобільного додатку ви будете використовувати технологію MAUI (Multi-platform App UI) з використанням мови програмування C#. Ця технологія дозволить створювати крос-платформові додатки для різних мобільних платформ з використанням одного коду.
-	Пристрій IoT: Для реалізації IoT-частини вашого проекту ви будете використовувати Raspberry Pi з використанням мови програмування C#. Це дозволить вам підключати різноманітні датчики та пристрої для збору даних і керування об'єктами на гірськолижному курорті.
